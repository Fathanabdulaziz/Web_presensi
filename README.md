# Website Presensi Sederhana

Website presensi online yang dibuat menggunakan HTML, CSS, dan JavaScript saja.

## Struktur Proyek

```
Web_presensi/
├── index.html          # Halaman login utama
├── dashboard.html      # Halaman dashboard presensi
├── css/
│   └── style.css       # Styling untuk website
├── js/
│   └── script.js       # Logika JavaScript
└── assets/             # Folder untuk gambar dan asset lainnya
```

## Fitur

- **Login Sederhana**: Sistem login dengan username dan password
- **Check In/Out**: Tombol untuk mencatat waktu masuk dan keluar
- **Riwayat Presensi**: Menampilkan daftar riwayat presensi
- **Penyimpanan Lokal**: Data disimpan di localStorage browser
- **Responsive Design**: Tampilan yang menyesuaikan dengan berbagai ukuran layar

## Cara Menjalankan

1. Buka file `index.html` di browser web
2. Login dengan username dan password apa saja (simulasi)
3. Gunakan tombol Check In dan Check Out untuk mencatat presensi
4. Lihat riwayat di bagian bawah dashboard

## Teknologi yang Digunakan

- **HTML5**: Struktur halaman web
- **CSS3**: Styling dan layout responsif
- **JavaScript (ES6+)**: Logika aplikasi dan interaksi

## Catatan

Website ini menggunakan localStorage untuk penyimpanan data, sehingga data akan hilang jika browser di-clear. Dalam aplikasi nyata, sebaiknya gunakan database dan server backend.
